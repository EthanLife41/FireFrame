Changed your .env or config? Restart the FireFrame server in your terminal (Ctrl-C, then run it again), then tap Reload App here. For Fully Kiosk, enabling fullscreen in the browser's own settings is the most reliable.

Restart FireFrame

FireFrame can't restart its own server safely, so do it from the Mac:

  1. In the Terminal window running FireFrame, press Ctrl-C.
  2. Run your start command again (e.g. ./fireframe.command).
  3. Back here, tap Reload App in Settings.
